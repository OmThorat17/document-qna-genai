# document-qna-genai

# Document Q&A using GenAI (RAG)

## Project Overview
This project allows users to upload PDF documents and ask questions.
The system generates answers using OpenAI based only on the uploaded document.

## Tech Stack
- Backend: FastAPI (Python)
- LLM: OpenAI GPT
- Vector Database: FAISS
- Frontend: React
- Architecture: Retrieval Augmented Generation (RAG)

## Features
- Upload PDF
- Ask questions
- Accurate answers from document
- Scalable industry architecture

## Future Scope
- Multiple document support
- Authentication
- Cloud deployment
